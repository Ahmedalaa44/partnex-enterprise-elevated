# Partnex CMS - Client Documentation

Welcome to your new Partnex content management system! This guide will help you manage all content on the Partnex website through Sanity CMS.

## Quick Start

### 1. Create Your Account

When you receive an invitation email from Sanity:
1. Click the **"Join project"** button in the email
2. Choose your preferred login method (Google, GitHub, or Email)
3. Complete the registration process
4. You'll be directed to the Sanity Studio dashboard

### 2. Access the Studio

Once your account is created, you can log in at:
```
https://sanity.io
```

Or directly access your project studio after it's deployed.

## Managing Content

### Hero Section (Main Banner)
This is the first thing visitors see on your website.

**What you can edit:**
- **Eyebrow text** - Small text above the main headline (e.g., "System Integration · Cloud · Cybersecurity")
- **Main headline** - Large title text
- **Highlighted word** - Single word that appears in gradient color
- **Subtitle** - Description text
- **Call-to-action buttons** - Primary and secondary buttons with text and URLs
- **Statistics** - Up to 4 stats shown in the hero (number + label)
- **Background image** - Hero section background

**How to edit:**
1. Click "Hero Section" in the sidebar
2. Update any field
3. Click "Publish" when done

### Services (Your Service Offerings)
List all services you provide with descriptions.

**What you can edit:**
- **Service title** - Name of the service (e.g., "Cloud Solutions")
- **Description** - 2-3 sentence description of the service
- **Icon** - Choose from available icons (Cloud, Shield, Server, etc.)
- **Display order** - Number to control the order services appear

**How to edit:**
1. Click "Services" in the sidebar
2. Click on a service to edit it, or click "+" to create a new service
3. Fill in the details
4. Click "Publish"

**Tips:**
- You can add, edit, or delete services anytime
- Use the display order number to rearrange them (lower numbers appear first)
- Keep descriptions concise - 2-3 sentences works best

### Partners (Your Technology Partners)
Add and manage technology partner logos.

**What you can edit:**
- **Partner name** - Company name (e.g., "Microsoft", "Cisco")
- **Logo image** - Upload the partner's logo
- **Website URL** - Link to partner's website
- **Display order** - Control the order logos appear

**How to edit:**
1. Click "Partners" in the sidebar
2. Click on a partner to edit or click "+" to add a new partner
3. Upload logo image (PNG, JPG, or SVG work best)
4. Click "Publish"

**Logo requirements:**
- Square or rectangular format
- At least 300x300 pixels
- PNG or JPG format
- Size: under 500KB

## Publishing Content

### Save vs. Publish
- **Draft** - Changes saved but not visible on website
- **Published** - Changes visible to all website visitors

### How to Publish
1. After making changes to any content
2. Click the **"Publish"** button in the top right
3. Confirm your changes
4. Changes appear on the website within seconds

### Scheduling (Advanced)
You can schedule content to publish automatically:
1. Click the dropdown next to "Publish"
2. Select "Schedule"
3. Choose the date and time
4. Content will automatically publish at that time

## Important Tips

### Best Practices
✅ **Do:**
- Keep descriptions clear and concise
- Use proper spelling and grammar
- Test links before publishing
- Upload images in web-friendly formats (PNG, JPG)
- Maintain consistent branding

❌ **Don't:**
- Leave required fields empty
- Use too many special characters
- Upload extremely large image files
- Publish without reviewing changes
- Change structure without permission

### Image Guidelines
- **Hero background**: 1920x1080px or wider
- **Partner logos**: Square format, 300x300px minimum
- **OG image**: 1200x630px (for social sharing)
- **Team photos**: 400x400px minimum
- **Project images**: 800x600px or wider
- File size: Keep under 2MB each

### SEO Tips
- Use clear, descriptive titles (50-60 characters)
- Include relevant keywords in descriptions
- Keep meta descriptions 150-160 characters
- Use high-quality OG images for social sharing
- Update when major changes occur

## Troubleshooting

### I don't see my changes on the website
- Did you click "Publish"? Changes must be published to appear.
- Try refreshing the website (press F5 or Ctrl+Shift+R)
- Wait a few seconds for the cache to update

### The upload button is greyed out
- You may not have permission to upload images
- Contact support if you need access

### I accidentally deleted content
- Click the "History" button to view previous versions
- You can restore a previous version if needed

### Content doesn't look right
- Check the formatting (bold, italics, links)
- Make sure you didn't accidentally add extra spaces
- Try republishing the content

### I can't login
- Check that you're using the correct email address
- Reset your password on the Sanity login page
- Check your spam folder for invitation email

## Advanced Features

### Media Library
All images are stored in the media library:
1. Click the image icon in the top menu
2. Browse or search for images
3. Click to use in your content

### Document History
See all changes made to any document:
1. Open any document
2. Click "History" in the top menu
3. Click any version to preview
4. Click "Restore" to revert to that version

### Search
Find any document quickly:
1. Use the search box in the left sidebar
2. Type what you're looking for
3. Click to open the document

### Keyboard Shortcuts
- `Cmd/Ctrl + S` - Save document
- `Cmd/Ctrl + Enter` - Publish
- `/` - Open command menu
